    Run task-level inference¶

    Choose a biological endpoint, pass in RNA or DNA sequences, and get task-level predictions through a standard pipeline backed by a documented checkpoint, model card, citation, and license.

    Fine-tuning

    Fine-tune a pretrained model¶

    Fine-tune a pretrained molecular model on a Hugging Face dataset or labelled local table. MultiMolecule maps sequence and label columns into trainable batches, then keeps training and evaluation in the runner.

    Python
     1
 2
 3
 4
 5
 6

    import multimolecule as mm


config = mm.Config(
    pretrained="multimolecule/ernierna",
    data={
        "root": "multimolecule/chanrg",
        "feature_cols": ["sequence"],
        "label_cols": ["secondary_structure"],
    },
)

runner = mm.Runner(config)
runner.train()

model = mm.AutoModelForRnaSecondaryStructurePrediction.from_pretrained(
    "multimolecule/ernierna-ss",
)

inputs = tokenizer("AUCAGCCUUCGUUCUGUAAACGG", return_tensors="pt")
outputs = model(**inputs)
